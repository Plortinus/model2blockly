Generation Pipeline

StepArtifactMeaning
1. Source model.model2blocklyThe selected model is adapted into the shared generator-facing model.
2. Intermediate EMF modelEditorSpecBoth the DSL and Ecore routes normalize into the same model shape.
3. Intermediate XMIintermediate/AppMaker_blocklyspec.xmiThe generator writes this XMI, reloads it, and validates the reloaded model.
4. Blockly editorhtml/*HTML, JavaScript, validation runtime and report files are emitted from the reloaded EditorSpec model.
