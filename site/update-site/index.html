Eclipse Update Site

Use this p2 repository URL in Eclipse: https://plortinus.github.io/model2blockly/update-site/

Install guide Troubleshooting Release checklist

Installation

  1. Open Eclipse.
  2. Select Help -> Install New Software....
  3. Add the update site URL above as a software site.
  4. Select the Model2Blockly feature and complete the installation wizard.

Requirements: Eclipse running on JDK 21, with dependency resolution enabled so Xtext and EMF can be installed from the configured Eclipse update sites.

If the install dialog shows no items, reload the site and clear Group items by category. Browser directory listings are not required; Eclipse reads content.jar and artifacts.jar directly.

If Model2Blockly is already installed and a newer build is not visible, clear Hide items that are already installed, remove stale Model2Blockly entries from Manage..., add this site again, and reload.
